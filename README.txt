研究室における自由課題用プログラム
自由課題とはDATK（マイコン）とモーター2つ，画像出力（openGLまたはUnity）を用いて創作物を作るVR研での課題である．
創作物は触覚フィードバックがあり，それに連動した画像出力があれば何でもよい．（IVRCを参考）

このプログラムは私がこの自由課題において”金魚すくい”を作成した際のものである．
＜FreeProject＞
Unityプロジェクト．
